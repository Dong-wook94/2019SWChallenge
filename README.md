# 2019SWChallenge <TechStudy>
- 2019 SW 챌린지 공모전 Team '소리가 보인다' 저장공간입니다.
- 이 브랜치는 프로젝트에 필요한 기술들을 공부하고 정리하며 프로젝트에 사용 가능한 소스코드를 만들기 위한 곳입니다.
- 각자 독립적으로 연구할 수 있는 공간을 각자의 디렉토리로 만들어서 자유롭게 연구/개발 하시면 됩니다!
- 하다가 모르는 것이 있거나 궁금한 것이 있다면 issue를 적극적으로 활용합시다!!
